# Flask Showcase Website

This folder contains a faculty-facing showcase website for the `Voice Typing Assistant` project.

## Stack

- Python
- Flask
- HTML
- CSS
- JavaScript

## Run

```powershell
cd "C:\Users\naksh\Desktop\Python project\voice_typing_assistant\flask_showcase"
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python app.py
```

Then open:

`http://127.0.0.1:5000`

## Notes

- This is a presentation website, not the desktop assistant itself.
- The site is designed to showcase the Python voice typing project to faculty.
- The visual direction is based on the floating voice assistant concept, with a premium compact look instead of a generic student dashboard.
